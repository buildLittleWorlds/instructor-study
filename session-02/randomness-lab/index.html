Where does randomness enter?

Conway's Game of Life can look wild, but with the same starting board and the same rule it always follows the same future. Use this lab to separate three things that are easy to blur together: the starting state, the update rule, and randomness during execution.

When a system surprises us, did the surprise come from its input, its rules, or randomness while it ran?

Change the starting state

New random board changes the input but keeps B3/S23 fixed. Different beginnings can produce different worlds without changing the system.

2

Add randomness while it runs

The noise control flips cells only after the normal update. That creates a deliberately noisy cellular automaton. It is useful for comparison, but it is not standard Conway.

3

Change the rule

This lab does not change the rule. If we replaced B3/S23, we would be studying a different cellular automaton—not merely a more random run of the same one.
