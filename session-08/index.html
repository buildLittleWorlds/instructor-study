Strongest pattern

Across eight prompts, directed deviation beat both the default and the random condition by about … points out of 20 — and the win survives every outlier toggle and both ways of averaging.

What it does not prove — write yours

Mine: It does not prove that directed prompting makes writing excellent — it proves that one small rater-set, using one rubric, preferred it on eight prompts; and if the only thing you prize is surprise, randomness is still the champion.

Method

  • Evidence: the 24 analysis rows (8 prompts × 3 conditions) are fetched, unchanged, from last week's file — ../session-07/data/dataset.json. This page ships no new rows and edits none, so the analysis cannot quietly touch the evidence you already explored.
  • Default path: the flagged drift row is excluded (including it flatters my favorite condition — the exclusion rule was set last week, before any average existed), and scores are aggregated by simple mean.
  • Everything is computed in your browser at load. Every number in the prose below is injected from that same computation — nothing is typed in by hand, so the words cannot disagree with the data.
  • Data honesty: the dataset's own note appears here when it loads.
  • The menu is authored too: I chose which three rows earned toggles and which five yardsticks made the list. See the limitation.

Interpretation

The true swerve won, and not by a hair: … for directed against … for default and … for random, out of 20. Notice what averaging did to last week's temptations. Random's one dazzling row — the piano-tuner line that earned it real envelope votes — is still there, still dazzling, and it changes the outcome not at all: a lucky row can win a sort, but it cannot win a mean. Directed's weak row didn't sink its condition either. Toggle the outliers above and the bars barely breathe. That is what a robust finding feels like under your hands.

But one path flips. Judge by earned surprise alone and random edges directed, … to …. That flip is not an embarrassment to the study — it is the study's point in different clothes. Surprise is the one thing randomness manufactures for free. What it cannot manufacture is surprise that fits: on fit, form, and return, the random condition finishes last on every path there is. The swerve only counts as excellence when it comes with a job to do.

Limitation. The rows this page analyzes are still authored, illustrative stand-ins for a real run (their own file says so above). Eight prompts and one rater-set is a pilot, not a verdict. Averaging 1–5 rubric scores treats judgment as arithmetic — defensible, but a choice, which is exactly why the mean/median switch exists. And the eighty paths are my eighty: I decided which rows deserved toggles and which yardsticks made the menu. An analysis that lets you take every path still starts from someone's map.

Try it yourself

Before you believe your own finding this week, make it walk three paths: drop your best row — does it hold? Drop your worst — does it hold? Swap mean for median — does it hold? Then write the two sentences that must always travel together: your strongest pattern, and what it does not prove. Use your AI as the skeptic, never the author: ask it "what does my data NOT prove?" and keep its best objection in your own words.
